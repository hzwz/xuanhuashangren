冻死不攀缘，
饿死不化缘；
穷死不求缘，
随缘不变，不变随缘，
抱定我们的三大宗旨。
舍命为佛事，
造命为本事，
正命为僧事，
即事明理，
明理即事，
推行祖师一脉心传。

人身上的蛆虫
_______________________________________________________________________________________
有了贪心、_心、痴心，就生出贪欲、_恚、愚痴的虫子。

    《华严经》上说：在不可说诸佛报身上，各各现出不可说诸毛孔。在每根毛孔中，又现出诸佛x土，又现出大宝莲华，又现出不可说的莲华叶。在每片叶中，皆放光明。在光明之中，又放光明，现出种种色相，无穷无尽。在光中现月，月中复现月，这种境界，都是诸佛所现。

    我们众生的身上，也有所现。现什么？现出虫子。在我们身上有无量的虫子，这虫子比厕所的蛆还要臭。在每条虫子的身上，又现出不可说的细菌。这种细菌，凡夫的肉眼是看不见的，必须是证得天眼通的人，才能看得见 ( 现在特大号显微镜也看不见 ) 。

    这种细菌，非常肮脏。不过，我们自己不知道而已。把大虫子 ( 身体 ) 看得那样的宝贵，那样的重要，又怕饿着，又怕冻着，又怕热着，想尽办法来保护这条大虫子。为什么这样的爱它？因为有三毒之心。有了贪心，就生出贪欲的虫子。有了_心，就生出_恚的虫子。有了痴心，就生出愚痴的虫子。总而言之，有了什么思想，就生什么虫子。

    我们的思想，比厕所的蛆虫还要肮脏十倍，尽想些邪知邪见，没有一点正知正见。思想越是肮脏，越想要肮脏的东西，乃至造成肮脏的世界，染污的世界，所以变成五浊恶世。五浊恶世，是由我们不清净的妄想心所造成的。这个娑婆世界，并没有浊，而是我们把世界弄浊了。五浊就是劫浊、见浊、烦恼浊、众生浊、命浊。这五种浊，多么可怕呀！

    我们心中有不正当的念头，例如嫉妒的念头和障碍的念头，时时在作怪，可是自己还不承认，反而把它覆藏起来，以为人不知道。其实这种行为，比妖魔鬼怪还要丑陋十分。丑陋的程度，简直是等于放出臭气，熏到三十三天去，把玉皇大帝熏得掩鼻而逃。你们看！这股臭气厉害不厉害？可知其臭不可闻也。

    自己的思想和行为，就是这样的臭，自己还不知道，认为自己是清高的人，那知把世界都给染污了。因此，变成五浊恶世。各位要知道，我们一念为正，就能补助天地之正气；我们一念是邪，就是臭气冲天，令天地容纳不了。这不是讲笑话，更不是戏论，的的确确是这样子。为什么天旱不雨？就因为人所放的臭气，冲到虚空中，把兴雨的龙，给熏跑了，无法执行的任务。
