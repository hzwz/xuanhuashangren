冻死不攀缘，
饿死不化缘；
穷死不求缘，
随缘不变，不变随缘，
抱定我们的三大宗旨。
舍命为佛事，
造命为本事，
正命为僧事，
即事明理，
明理即事，
推行祖师一脉心传。

虔诚念吉祥神咒
_______________________________________________________________________________________
为令全世界人类消弭灾难，都得到吉祥。

    从今年元旦 ( 春节 ) 开始，大家诚心来念消灾吉祥神咒：「曩谟。三满多。毋驮喃。阿钵底。贺多舍。娑曩喃。怛侄他。。怯怯。怯四。怯四。恕Ｈ胪。入瓦。钵入瓦。钵入瓦。底瑟g。底瑟g。瑟致哩。瑟致哩。娑j咤。娑j咤。扇底迦。室哩曳。娑瓦诃。」若能专一其心来念，一定会有不可思议的感应。所谓「感应道交」，会获得这种境界。

    我们为什么要念消灾吉祥神咒？为令全世界人类消弭灾难，都得到吉祥。这是万佛圣城举行消灾吉祥法会的动机。总而言之，是为拯救众生出离苦海而念，是没有其它企图而念。一般道场，无论做什么法会，一定要宣传，找位功德主 ( 出钱做功德的人 ) 。可是我们万佛圣城与众不同，不管做什么法会，向来没有功德主，只有主功德。什么叫主功德？就是万佛圣城主持这种功德，然后把这种功德，回向给全世界上所有的人类。

    我们在这里，诚心诚意的念消灾吉祥神咒，帮助所有的人类获得利益，不要只为万佛圣城这几十个人念的。这个法会，大而无外，没有一个众生能超出这个法会之外。无论信佛或不信佛，无论是善人或是恶人 ; 我们都为他们回向，令其离苦得乐。我们做这种法会，不求任何代价，只是尽修道人的本分，也可以说尽天职，应当为一切众生求安乐。如果不认真做，那就失去真正的意义了。所以要至诚恳切的念，感动佛菩萨大发慈悲心，怜愍我们众生，将灾难减轻。或者将大的灾难化为小的灾难，将小灾难化为乌有。这是我们尽了一分力量，为全人类祈祷，消灾吉祥。这是有价值的事！

    我们借着万佛圣城这样优越的道场，来为全世界全人类消灾难，最为理想。如果没有这个好地方，就想作功德，也没有地方可作。所以，万佛圣城规定在每年阴历除夕之夜洒净，大年初一之晨，开始念消灾吉祥神咒。万佛圣城作为功德主，也就是主办这种功德，回向给全世界所有的众生。

各位！参加这种法会是多么光荣！多么幸运！应该打起精神来，诚心的念，不可敷衍了事，躲懒偷安。大家团结一致为拯救众生而念！念！念！念！
