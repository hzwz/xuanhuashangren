冻死不攀缘，
饿死不化缘；
穷死不求缘，
随缘不变，不变随缘，
抱定我们的三大宗旨。
舍命为佛事，
造命为本事，
正命为僧事，
即事明理，
明理即事，
推行祖师一脉心传。

正法住世的道场
_______________________________________________________________________________________
万佛圣城是正法住世的道场，是十方诸佛菩萨拥护的道场。

    万佛圣城是正法住世的道场，是世界佛教发源地，也是世界佛教归源处。有了万佛圣城所在地，天魔外道，吓得发昏，恐怖毛竖。旁门左道，吓得发抖，战战兢兢。他们寝不安席，食不知味，所以乃用尽心机，想尽办法来破坏万佛圣城、消灭万佛圣城，令万佛圣城不成功，令万佛圣城不存在。这种想法，等于痴人说梦，是不可能的事。

    万佛圣城是十方诸佛拥护的道场。所谓「从地涌出，现宝王x。」所以万佛圣城坚固如金刚，任何人破坏不了，也消灭不了。换言之，万佛圣城就是世界佛教的灯塔，照亮了芸芸众生黑暗的心，现出清净光明心，向菩提觉道迈进。

    凡是住在万佛圣城的人，都有大善根、大德行。否则是住不下来的。有德行的人，来到万佛圣城，如同入宝山一样，获得无价之宝 -- 明心见性。没有德行的人，来到万佛圣城，如同坐针毡一样，片刻不安。

    万佛圣城在这个地球是开始弘扬正法的道场，所以住众们一举一动、一言一行，皆尊守佛制。但在妖魔鬼怪的眼中，都认为不对。万佛圣城每日吃一餐 ( 日中一食 ) ，夜不倒单 ( 胁不着席 ) ，时时搭衣 ( 袈裟不离身 ) 、男女禁语 ( 互不讲话 ) 。外面的人认为这是落伍，不合潮流，为什么？因为万佛圣城提倡正法，而他们却喜欢末法时代的生活，一切不受限制。所以万佛圣城所行所做，他们认为不对，故极力攻击、破坏、消灭。

    有很多人，到万佛圣城来，觉得周身不自在。因为在世间混惯了，到了万佛圣城觉了格格不入，而不欢喜万佛圣城的清规。出家人的苦修，在他们的眼中看来，是违背自然。他们不知道：「吃苦是了苦，享福是消福。」的明训，只图一时之安乐，种来生之业报。

    万佛圣城不会同流合污，万佛圣城的人不作乡愿。凡在万佛圣城的人，皆发大菩提心，努力用功修道。所以万佛圣城无论是打地藏七、或观音七、或佛七、或禅七，都是特别辛苦，没有休息。想出家修道，却不能吃苦，又想自在，又想享受，又想开悟，又想成佛 -- 根本没有这种道理。

    世间上没有不劳而获的事。样样想占便宜，那是痴人妄想。所谓「一分耕耘，一分收获。」这是至理名言。万佛圣城的住众，皆明白这种道理，所以人人努力，个个认真，用功办道，发菩提心。一天到晚精进，无暇打妄想。根本也不想去攀缘，不和任何人拉关系。凡是来到万佛圣城的人，对于这一点，应该认识清楚。

    万佛圣城的一草一木，旁门外道也破坏不了。为什么？因为邪不胜正的缘故。万佛圣城是正法住世的道场，是十方诸佛菩萨拥护的道场。所以万佛圣城始终是胜利者，经得起大风暴，永远存在的。别有用心的人，真是枉费心机。
